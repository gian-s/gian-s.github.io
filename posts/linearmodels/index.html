Facebook Business Cosmetics

Linear Regression Analyses on Facebook Cosmetics site's metrics
December 2020 · 1 min · Boston University | Edit
Table of Contents

  • Links

Summary#

Semester-long project worked on a team of for Linear Models course at Boston University MA575. The project consisted of performing various analyses using models and techniques we learned in class. The task was to use different regression models and techniques in order predict the number of user interactions for a cosmetics business’ Facebook site at a given time.

Role:#

I was tasked with a group to create in-depth analyses of data which was mined from an online business’ Facebook page. My team and I were given the freedom to use any of the analytical tools taught to us in class as well as research our own methods to create our models. I was the lead programmer in my team and was tasked with using the R programming language to code all of the models, diagnostics, and helpful visualizations needed for our project. I was also able to implement my own R function of an iterative algorithm we were taught in class for finding significant variables in our data. The final project involved presenting our semester long findings and the methodologies used to our classmates and professors.
